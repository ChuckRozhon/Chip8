Chip8
=====

Usage: Chip8 'chip8application' 

Keyboard: 

This is the definition of the Chip8 keyboard used

    Keypad                   Keyboard
    +-+-+-+-+                +-+-+-+-+
    |1|2|3|C|                |1|2|3|4|
    +-+-+-+-+                +-+-+-+-+
    |4|5|6|D|                |Q|W|E|R|
    +-+-+-+-+       =>       +-+-+-+-+
    |7|8|9|E|                |A|S|D|F|
    +-+-+-+-+                +-+-+-+-+
    |A|0|B|F|                |Z|X|C|V|
    +-+-+-+-+                +-+-+-+-+

You can find some games that run on Chip8 at http://www.zophar.net/pdroms/chip8/chip-8-games-pack.html
